html5 e css

A HTMLé uma linguagem de marcação de para a criação de sites. Atualmente na versão 5, ela usa semântica dos elementos para criar o conteúdo de uma pagina Web. Com as CSS criamos as configurações visuais que um site vai ter.

JavaScript

JavaScript é uma linguagem de script criada para aumentar a interatividade entre um site e seus visitantes. Os códigos em JS são executados dentro do navegador e dão acesso aos elementos HTML por meio do DOM.
